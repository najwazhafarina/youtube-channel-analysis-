# YouTube Channel Analysis

## Overview

This project analyzes the Top 1000 YouTube Channels dataset to understand patterns in subscriber distribution, categories, and video views.

The analysis was done using Python for data cleaning and exploration, and Power BI for dashboard visualization.

## Tools

* Python
* Pandas
* Matplotlib
* Seaborn
* Power BI

## Dataset

Top YouTube Channels dataset containing information about subscribers, video views, categories, and countries.

## Key Insights

* Entertainment and Music dominate the most subscribed YouTube channels.
* Subscriber distribution is highly skewed.
* Channels with more subscribers tend to generate more video views.

## Project Structure

* `data/` → dataset files
* `scripts/` → Python analysis code
* `dashboard/` → Power BI dashboard
